Usage: Driver.jars are located into the res folder

Once navigated through the correct folder on a cmd window
the drivers can be run

Usage MelodyDriver
java -jar MelodyDriver.jar frequency

where frequency is the number between 0-13k inclusive
and the value can have up to 3 decimal places

usage AppDriver
java -jar AppDriver file.txt

where the file exists
and the file contains notes separated by ","
